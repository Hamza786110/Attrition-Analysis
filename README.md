# Attrition Analysis

## Description  
This project focuses on analyzing employee attrition data using SQL and basic data analysis techniques. The aim is to understand why employees leave an organization and to find patterns related to departments,
performance, and other factors.

## Tools Used  
- SQL  
- Excel / CSV Data  
- Dashboard / Visualizations  

## What I did in this project  
- Analyzed employee attrition data  
- Calculated attrition rates  
- Compared attrition across departments  
- Created insights to understand employee turnover  

## How to use this project  
1. Clone the repository  
2. Open the SQL files and run them in your SQL tool  
3. Check the results and dashboard files
